A Django 5 application with signup, login, logout, forgot password, reset password, and a home page. The system supports MySQL as the database and can send real emails via SMTP. It also includes a modern interactive frontend with Bootstrap and scroll animations.

Features

User registration with validation

Login/logout functionality

Forgot password with email reset code

Password reset form

Home page protected by login

Interactive, animated interface using Bootstrap & AOS

MySQL database backend

Optional real email sending via SMTP (Gmail, Outlook, etc.)

Requirements

Python 3.12+

Django 5.2+

MySQL 8+

mysqlclient package

A Gmail or Outlook account (for SMTP email)
